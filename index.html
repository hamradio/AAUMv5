AAUMv5 ANOTHER ADAM USAGE METER

AAUM is a Windows Vista Sidebar Gadget that displays usage information for customers of Adam Internet, a South Australian based ISP.

It is a free program that is easy to download & install and can be used by anyone with an Adam Internet account.

Download

AAUMv5.gadget

Features

  • Displays important account usage information, including peak; off-peak; uploaded & Usenet data
  • Coloured status bars give you a visual indication of how much data has been used in each category
  • Moving the mouse cursor over a status bar displays full textual usage information
  • Updates automatically every 30 minutes

Requirements

  • Microsoft Windows Vista or Windows 7, with Sidebar installed
  • An Adam Internet account
